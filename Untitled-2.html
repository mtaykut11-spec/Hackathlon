# Proje: Cloop-benzeri Ayrıntılı Site (ASP.NET Core Razor Pages) Bu proje Visual Studio için hazır bir ASP.NET Core Razor Pages (.NET 7+) şablonudur. Amaç: https://cloop.sg/ tarzı bir site oluşturmak; fakat içerik ve özellikler daha detaylıdır. --- ## Dosya yapısı ``` CloopDetailed/ ├─ CloopDetailed.csproj ├─ Program.cs ├─ appsettings.json ├─ Models/ │ └─ ContactModel.cs ├─ Pages/ │ ├─ _ViewImports.cshtml │ ├─ _ViewStart.cshtml │ ├─ Shared/_Layout.cshtml │ ├─ Index.cshtml │ ├─ Index.cshtml.cs │ ├─ Articles.cshtml │ ├─ Articles.cshtml.cs │ ├─ Article.cshtml │ ├─ Article.cshtml.cs │ ├─ Resources.cshtml │ ├─ Podcast.cshtml │ ├─ Contact.cshtml │ └─ Contact.cshtml.cs ├─ wwwroot/ │ ├─ css/site.css │ ├─ js/site.js │ └─ media/ └─ Controllers/ └─ ApiController.cs ``` --- ## Program.cs ```csharp using Microsoft.AspNetCore.Builder; using Microsoft.Extensions.DependencyInjection; using Microsoft.Extensions.Hosting; var builder = WebApplication.CreateBuilder(args); builder.Services.AddRazorPages(); builder.Services.AddControllers(); var app = builder.Build(); if (!app.Environment.IsDevelopment()) { app.UseExceptionHandler("/Error"); app.UseHsts(); } app.UseHttpsRedirection(); app.UseStaticFiles(); app.UseRouting(); app.MapControllers(); app.MapRazorPages(); app.Run(); ``` --- ## Pages/Shared/_Layout.cshtml ```html @ViewData["Title"] - Cloop Detailed
